turtle.clear()   清空turtle視窗，但是turtle的位置和狀態不會改變

turtle.reset()   清空視窗，重置turtle狀態為起始狀態

turtle.undo()    撤銷上一個turtle動作

turtle.isvisible()   返回當前turtle是否可見

stamp()    複製當前圖形


circle(50)   #整圓;

circle(50,steps=3)    #三角形;

circle(120,180)    #半圓
